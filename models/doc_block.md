This column contains the different trasitionary status of the order.

Definition of Status 
 - /placed - the order has been initiated
 - /shipped - the order has departed from the shipping facility and enroute to delivery address
 - /completed - order has been delivered
 - /return_pending - customer has initiated a return of the product
 - /returned - an initiated return has been completed
